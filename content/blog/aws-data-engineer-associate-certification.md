I'm happy to share that I have earned the **AWS Certified Data Engineer – Associate** certification.

The AWS Certified Data Engineer – Associate validates skills and knowledge in core data-related AWS services, including the ability to ingest and transform data, orchestrate data pipelines while applying programming concepts, design data models, manage data lifecycles, and ensure data quality.

You can learn more about this certification on the [official AWS page](https://aws.amazon.com/certification/certified-data-engineer-associate/).
